## Setup Locally (Windows / PowerShell)

Install Packages:

```bash
npm install
```

Create `.env` (copy from example):

```bash
copy .env.example .env
```

Run App:

```bash
npm run dev
```

## Setup Build

Install Packages:

```bash
npm install
```

Create `.env` (copy from example):

```bash
copy .env.example .env
```

Build:

```bash
npm run build
```

Run App:

```bash
npm run preview
```
